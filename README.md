# 💍 KalyanamUndo

> **Find the nearest wedding. Find the nearest biriyani. 🍛**

KalyanamUndo is a community-powered map that helps you discover weddings happening around you. Whether you're looking for a nearby marriage function, curious about what's happening in your area, or simply continuing the legendary tradition of **"Kummanadi"** (attending weddings without an invitation just for the fun of it 😄), this project is for you.

> ⚠️ **Disclaimer:** This project is built for fun and as a community experiment. Please always respect the privacy of families and behave responsibly at any event you attend.

---

## ✨ The Story

One random day, this idea popped into my head.

Growing up, my friends and I occasionally did something we jokingly called **"Kummanadi"**—walking into a wedding, enjoying the atmosphere, eating a delicious biriyani or sadhya, wishing the couple well, and leaving with great memories.

That made me wonder...

> **"What if there was a map showing weddings happening nearby?"**

And that's how **KalyanamUndo** was born.

---

## 🚀 What is KalyanamUndo?

KalyanamUndo is a map-based web application where the community can:

- 🗺️ **Explore Venues:** View convention centres and auditoriums across Kerala.
- 💍 **Live Weddings:** See weddings currently happening.
- 📅 **Upcoming Events:** Check upcoming wedding programs.
- ➕ **Contribute Venues:** Add missing convention centres to the map.
- 📝 **Update Details:** Add or edit wedding details for an existing venue.
- 🔍 **Search & Filter:** Search venues by name or filter them by district.

---

## 🎉 How It Works

1. **Open the website** and browse the interactive map of Kerala.
2. **Click on a convention centre marker** to see what's happening.
3. If a wedding is scheduled, you'll see details such as:
   - Bride 👰 & Groom 🤵
   - Date & Time 🕒
   - Food (optional 😋)
4. If no wedding is available, you can add one!
5. Missing a venue? You can add new auditoriums or convention centres for the community.

*Note: Once a wedding's end time has passed, the application automatically removes it from the "live" list to keep the map up-to-date.*

---

## 🟢 Marker Colors

- 🟢 **Green:** Wedding happening right now.
- 🟡 **Yellow:** Upcoming wedding.
- ⚪ **Gray:** No wedding added yet.

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Database:** PostgreSQL
* **ORM:** Prisma
* **Authentication:** Auth.js (NextAuth)
* **Maps:** React Leaflet & OpenStreetMap
* **Styling:** Tailwind CSS
* **Forms & Validation:** React Hook Form & Zod
* **Deployment:** Vercel
